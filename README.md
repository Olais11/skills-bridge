# Skills Bridge Tanzania

**Skills Bridge Tanzania** is a social initiative designed to bridge the gap between youth potential and employment opportunities in Tanzania.  
The platform promotes skills development, partnerships, and empowerment opportunities for young people through training, mentorship, and collaboration.

---

## Project Overview

This is a **static website** built using HTML and CSS.  
It includes pages highlighting the mission, vision, and initiatives of Skills Bridge Tanzania, as well as a contact form for inquiries and partnerships.

---

## Project Structure

```
skills-bridge/
│
├── index.html          # Homepage
├── mission.html        # Mission and vision details
├── about.html          # About section and organization background
├── contact.html        # Contact form and engagement opportunities
├── message.html        # Confirmation page after form submission
└── style.css           # Shared styling for all pages
````

---

## Features

- Responsive design for desktop and mobile devices  
- Contact form for user inquiries and partnerships  
- Clean, minimal layout with user-friendly navigation  
- Embedded media sections for videos and images  
- Consistent branding with unified header, footer, and color palette  

---

## How to Use

1. **Clone the repository**
```
git clone https://github.com/Olais11/skills-bridge.git
````

2. **Open the website locally**

   * Navigate to the project folder
   * Open `index.html` in your web browser

3. **Edit content**

   * Modify text or add new sections in the HTML files
   * Adjust styling in `style.css`

4. **Deploy online**

   * Host on GitHub Pages, Netlify, or Vercel
   * For GitHub Pages:
     Go to your repository → Settings → Pages → Source: main branch / root
